About


This website was made using Bootstrap 3, LESS, jQuery, and a little bit of native JavaScript. No BootStrap theme was used, and the full source code was utilized for easy integration with custom LESS files.

A jQuery plugin, MixItUp, was used for the image gallery animation effects and sorting system. Finally, one native JavaScript algorithm was used for all pages to make the self-updating timers. No HTML preprocessors were used. Individual icons and images, as well as the Mixtin logo, are not my own work. The sources for these assets can be found below.
